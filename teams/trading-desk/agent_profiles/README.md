# Agent Profiles

These files define the desk-level role descriptors for each trading-desk agent.

Included roles:
- `orchestrator_trader.yaml`
- `market_researcher.yaml`
- `strategy_agent.yaml`
- `risk_manager.yaml`
- `portfolio_monitor.yaml`

Use these as the team-layer mapping between the desk topology and the runtime profile/config layer under `profiles/`.
