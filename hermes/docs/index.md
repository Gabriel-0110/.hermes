# Hermes Docs

This documentation set is intentionally concise and operational.

## What Hermes Is

Hermes is a multi-agent cryptocurrency trading platform scaffold with a FastAPI backend, Next.js Mission Control frontend, PostgreSQL/TimescaleDB data layer, and model-provider flexibility across local and cloud inference endpoints.

## Documentation Map

- [Development](development.md): local setup, tooling, and environment variables
- [Agents](agents.md): agent roles and boundaries
- [Shared Intelligence Layer](shared-intelligence-layer.md): common data and tool abstractions
- [Mission Control](mission-control.md): operator-facing control plane concepts
- [Deployment](deployment.md): local orchestration and infrastructure notes

## Current Scope

The repository is in a foundation phase. It aims to provide a clean base rather than fake completeness. Placeholder modules and TODO markers indicate where future implementation is expected.
